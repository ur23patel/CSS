Default Setup

Defult setup of flex properties is flex: 0 1 auto. Thats shorthand for flex-grow . flex-shrink . and flex-bassis properties values.
This setup basically tells us:"All the flex children should not grow.should shrink at the same rate and have their initial size based on their width or height"
