Css Syntax

CSS comprises style rules that are interpreted by the browser and then applied to the corresponding elements in your document. A style rule set consists of a selector and declaration block.

Selector -- h1 Declaration -- {color:blue;font size:12px;}

  • The selector points to the HTML element you want to style.
  • The declaration block contains one or more declarations separated by semicolons.
  • Each declaration includes a CSS property name and a value, separated by a colon.
    For Example:

    color is property and blue is value.
    font-size is property and 12px is value.
  • CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.

There are 3 ways to write CSS in our HTML file

  1. Inline (not recommanded)
  2. Internal (not prefrabble/not easy to apply in multiple pages)
  3. External (recommanded, prefrabble and correct way to apply css)
  • Inline > Internal > External

Inline Css

Within tag wit the help of style attribute

  • Before CSS this was the only way to apply styles
  • Not an efficient way to write as it has a lot of redundancy
  • Self-contained 
  • Uniquely applied on each element
  • The idea of separation of concerns was lost

opening tag style="prop:value;"

<h2 style="color: rgb(114, 165, 42);font-size: 56px;">Inline CSS Example</h2>

Internal Css

Within page in head sec prefferable applied for all selectors which is definded style tag will help to define internal css

  • With the help of style tag, we can apply styles within the HTML file
  •  Redundancy is removed

External CSS

recommanded by W3C

mainly we use it because it will applied for multiple pages

  • With the help of <link> tag in the head tag, we can apply styles
  • Reference is added 
  • File saved with .css extension
  • Redundancy is removed
  • The idea of separation of concerns is maintained
  • Uniquely applied to each document
