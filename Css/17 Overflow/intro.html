overflow:

This property can be specifies weather tp clip the content or add to scrollbars when the content of an element is too big to fit in the specified area.

Overflow Property:

    /* Keyword values */
  • overflow: visible;
  • overflow: scroll;
  • overflow-y: scroll;
  • overflow-x: scroll;
  • overflow: hidden;
  • overflow: clip;
  • overflow: auto;
  • overflow: hidden visible;
    /* Global values */
  • overflow: inherit;
  • overflow: initial;
  • overflow: revert;
  • overflow: revert-layer;
  • overflow: unset;
